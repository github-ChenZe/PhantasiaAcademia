\subsubsection{Elementary Group Theory} % (fold)
\label{ssub:elementary_group_theory}

Let $G_1$ and $G_2$ be two Abelian groups. A map $\func{f}{G_1}{G_2}$ is said to be a homomorphism if
\[ f\pare{x+y} = f\pare{x} + f\pare{y} \]
for all $x,y \in G_1$. If $f$ is also a bijection, $f$ is called an \gloss{isomorphism}.
